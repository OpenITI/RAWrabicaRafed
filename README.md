# Rafed.net scrape

date: 2020-10-09

This repo contains the files scraped from the Rafed.net website.

In `books`, you'll find the html version of the Rafed books.

The folder `subject_pages` contains the list of books with their metadata for each subject in the collection: 

  1. القرآن وعلومه
  2. الحديث وعلومه
  3. العقائد والكلام
  4. الفقه
  5. أصول الفقه
  6. رجال الحديث
  7. سيرة النبي (ص) وأهل البيت (ع)
  8. التراجم
  9. الأخلاق
  10. الفرق والمذاهب
  11. التاريخ والجغرافيا
  12. الفلسفة والمنطق
  13. القانون
  14. الإقتصاد
  15. العلوم السياسية
  17. علم‌النفس والتربية والاجتماع
  18. العلوم الطبيعيّة
  19. الشعر والأدب
  20. العرفان والأدعية والزيارات
  21. دليل المؤلفات
  22. العامة
  23. الطّب
  24. اللغة والبلاغة
  26. مجلّة تراثنا

The metadata extracted from these pages can be found in `meta.json` and `meta.csv`.
